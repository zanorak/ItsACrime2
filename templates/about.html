JSON means JavaScript Object Notation. It is a data format written in plain text that can be read and written by computers and humans. It's organized as a series of name, value pairs. Many different types of data can be represented in this format such as “object, record, struct, dictionary, hash table, keyed list, or associative arrays.” The name implies that it was originally written to support JavaScript, utilities to read and write JSON data exist in most programming languages. The data files come from CORGIS Project (The Collection Of Really Great, Interesting, Situated Datasets). The site provides a large variety of datasets in different formats specifically for use in programming assignments.

The data comes from the FBI's Uniform Crime Reporting Program. The FBI has been collecting crime statistical data from more than 18,000 sources since 1930. They do this to provide reliable and consistent data to law enforcement, law makers, and researchers. Data is gathered from the National Incident-Based Reporting System (NIRBS), Summary Reporting System (SRS), The Law Enforcement Officers Kill and Assaulted Program (LEOKA), and Hate Crime Statistics.

The data in this data set covers crime arranged by state and includes the years from 1960 to 2019. The data is presented as both Rates per 100,000 persons, and total numbers. The crimes are divided into two categories. Property crime that refers to burglary, larceny, and motor related crime. And violent crime refers to assault, murder, rape, and robbery.
